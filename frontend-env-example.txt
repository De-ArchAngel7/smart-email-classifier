# Smart Email Classifier Environment Setup Guide

## Backend Environment Variables (.env)
HF_API_TOKEN=your_hugging_face_token_here
PORT=8000
HOST=0.0.0.0

## Frontend Environment Variables (frontend/.env.local)
NEXT_PUBLIC_API_URL=http://localhost:8000

## Production Environment Variables
# Backend
HF_API_TOKEN=your_actual_hugging_face_token
PORT=8000

# Frontend
NEXT_PUBLIC_API_URL=https://your-backend-url.com

## Deployment Platforms Environment Setup:

### Railway/Render (Backend)
- HF_API_TOKEN: your_actual_hugging_face_token
- PORT: (automatically set by platform)

### Vercel (Frontend)  
- NEXT_PUBLIC_API_URL: https://your-backend-url.com

## How to Get Your Hugging Face Token:
1. Go to https://huggingface.co/settings/tokens
2. Create a new token with "Read" permissions
3. Copy the token and set it in your environment variables
